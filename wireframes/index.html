Catatan Teknis Wireframe

  • Semua file berdiri sendiri (standalone) — tidak ada dependensi eksternal kecuali ikon Unicode
  • CSS custom properties: --primary #1E3A5F, --accent #F5A623, --annotation #2979FF
  • Tombol toggle Desktop/Mobile di setiap halaman untuk preview responsif
  • Anotasi berwarna biru miring menjelaskan komponen dan keputusan UX
  • Placeholder gambar menggunakan div dengan background #E0E0E0 dan teks dimensi
  • Konten bahasa Indonesia realistis, bukan Lorem Ipsum
  • Semua halaman mencakup: navbar sticky, floating WhatsApp button, dan footer 4 kolom
  • Breakpoints: mobile <768px, tablet 768-1024px, desktop >1024px
